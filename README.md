# dataapi
Une api qui permet de stocker des articles dans un ficher json

# MAIS COMMENT SA FONCIONNE ?
C'est super simple, vous pourrez récupèrer vos articles dans un format json avec une requête GET /news/get 
Et pour en ajouter il suffit de faire une requête POST /news/post

# EN QUOI C'EST UTILE ?
Cela permet d'avoir son sur site la possibilité de mettre des articles. A vous ensuite de créer la partie publication :DD
